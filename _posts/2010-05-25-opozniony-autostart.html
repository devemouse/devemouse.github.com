W pracy mam powolny komputer. Denerwowało mnie to  że dużo aplikacji uruchamia się od razu po starcie systemu. Można było użyć zewnętrznej aplikacji, która ten start opóźni. Dodatkowo część aplikacji potrzebna jest tylko krótko po starcie systemu. Nie da się również uruchomić kilku programów po kolei używając prostych narzędzi dostarczonych z Windowsem. Dlatego wpadłem na pomysł rozszerzenia odrobinę mechanizmu zaplanowanych zadań z windowsa.

Prezentuję skrypt, który tworzy zaplanowane zadania relatywne do daty/godziny uruchomienia skryptu:  afterstart.rb.



Skrypt napisałem w moim ulubionym Rubym {% highlight ruby %} require 'highline/import' require 'ezcrypto' p "After Start" tasks = [ { :name => 'kill_dropbox', :command => "taskkill /f /im Dropbox.exe", :time => 20 }, { :name => 'kill_TSVNCache', :command => "taskkill /f /im TSVNCache.exe", :time => 5 }, { :name => 'start_Todolist', :command => "C:\Darek\\app\\todolist\\ToDoList.exe", :time => 5 }, { :name => 'backup_jazdy', :command => "copy k:\\Shared\\synowiecd\\jazdy.xls c:\\Darek\\Doc\\dojazd /Y", :time => 60 }, { :name => 'start_dropbox', :command => "C:\\Documents and Settings\\synowiecd\\Application Data\\Dropbox\\bin\\Dropbox.exe", :time => 7*60 }, ] pass=EzCrypto::Key.generate key = pass.encrypt ask("Enter your password: ") { |q| q.echo = "*" } current_tasks = %x[schtasks] tasks.each do |task| if current_tasks =~ /#{task[:name]}/ then system("schtasks /delete /tn #{task[:name]} /f") end system("schtasks /create /RU synowiecd /RP #{pass.decrypt(key)} /sc ONCE /tn #{task[:name]} /tr \"" + task[:command] + "\" /st " + (Time.now + (task[:time]*60)).strftime("%H:%M:%S") ) end pass = nil keu = nil current_tasks = %x[schtasks] puts current_tasks {% endhighlight %} Pozdrawiam Darek.
